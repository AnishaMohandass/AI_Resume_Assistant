# AI_Resume_Assistant

## AI Resume Assistant using RAG, LangGraph, Multi-Agent Systems, and LLMs.

Built an intelligent AI-powered Resume Assistant using
* Retrieval-Augmented Generation (RAG)
* LangGraph
* LangChain
* FAISS Vector Database
* Hugging Face LLMs
  
Designed a multi-agent workflow system for
* ATS score analysis
* Skill gap identification
* Resume improvement suggestions 

Implemented 
* document loading
* text chunking
* embeddings
* semantic search
* agent orchestration

to create an end-to-end Generative AI application.



## Sample Input Questions:
1) What are the missing skills in the resume?
2) What is the ATS score of the resume?
3) How to improve the resume professionally?
