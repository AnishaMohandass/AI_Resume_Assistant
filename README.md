# AI_Resume_Assistant
AI Resume Assistant using RAG, LangGraph, Multi-Agent Systems, and LLMs.

Built an intelligent AI-powered Resume Assistant using Retrieval-Augmented Generation (RAG), LangGraph, LangChain, FAISS Vector Database, and Hugging Face LLMs. 
Designed a multi-agent workflow system for ATS score analysis, skill gap identification, and resume improvement suggestions. 
Implemented document loading, text chunking, embeddings, semantic search, and agent orchestration to create an end-to-end Generative AI application.
